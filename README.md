# Mind Hacker

Mind Hacker is a browser-based puzzle hacking simulation built with plain HTML, CSS, and JavaScript.

## Play Online

Live game: https://mjalisto1111.github.io/mind-hacker-game/

If the link shows 404 right after deployment, wait a few minutes and refresh while GitHub Pages finishes publishing.

## Features

- Level-based progression
- Puzzle types:
  - Binary conversion
  - Logic gates (AND/OR)
  - Password decoding (Caesar shift)
- Per-level countdown timer
- Score system based on speed and accuracy
- Correct/incorrect feedback
- Restart mission option
- Responsive dark hacker-style UI

## How To Play

1. Read the puzzle prompt.
2. Enter your answer in the input box.
3. Press **Inject**.
4. If correct, continue to the next level.
5. If wrong or timer runs out, you lose points.

## Run Locally

Open `index.html` in any modern browser.

## Project Structure

- `index.html` - game layout
- `styles.css` - UI and animations
- `script.js` - puzzle logic, timer, scoring, level flow
